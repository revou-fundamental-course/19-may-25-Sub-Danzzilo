Temperature Converter







Explanation

To convert between different temperature scales, you can use the following formulas:

Celcius (°C)
  • Celcius to Fahrenheit: F = C x 9/5 + 32
  • Celcius to Kelvin: K = C + 273.15
  • Celcius to Reamur: Ré = C x 4/5
Fahrenheit (°F)
  • Fahrenheit to Celcius: C = (F - 32) x 5/9
  • Fahrenheit to Kelvin: K = (F - 32) x 5/9 + 273.15
  • Fahrenheit to Reamur: Ré = (F - 32) x 4/9
Kelvin (°K)
  • Kelvin to Celcius: C = K - 273.15
  • Kelvin to Fahrenheit: F = (K - 273.15) x 9/5 + 32
  • Kelvin to Reamur: Ré = (K - 273.15) x 4/5
Reamur (°Ré)
  • Reamur to Celcius: C = Ré x 5/4
  • Reamur to Fahrenheit: F = Ré x 9/4 + 32
  • Reamur to Kelvin: K = Ré x 5/4 + 273.15

Note: The formulas above are based on the definitions of the temperature scales.
